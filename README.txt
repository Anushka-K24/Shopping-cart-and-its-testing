README.txt file for Shopping Cart.


//Description - 
This folder contains a Shopping Cart project in vanilla javaScript. 
The test cases are written in jest, which are in the shoppingCart.test.js file.
The objective of the project is to add some items using their particular barcode in the cart,
calculate the total amount and check if the amount is according to the number of items in the cart.
Similarly, items can be removed from the cart and the total bill is generated.

//steps to run the program - 

1)Run npm install in the terminal for the first time after downloading the files.
2)Run npm test in terminal to run the tests.
